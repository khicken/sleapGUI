# sleapGUI
GUI application that automates specified commands for SLEAP.

## Installation
### Conda
This
```sh
conda install sleapgui-c
```

## Usage
To launch the application, open the executable or if installed through `pip` run:
```sh
mediagui
```

## Compatibility
| Platform | Python Version |
|----------|----------------|
| Windows  | 3.10 |
| macOS    | 3.10 |
| Linux    | 3.10 |
